# 情绪急救

## 引入

欢迎来到 `“花开如你”（Blossom Like You）冥想系列`。

也许你正在度过一段艰难的时光，我们将通过这个简短的联系，帮你从情绪的波动中，慢慢平复下来。

## 环境准备

寻找一个不被打扰的空间，慢慢的，安静的坐下来。当你准备好了。慢慢的闭上眼睛。

让我们先来放松身体，让牙齿慢慢松开。双肩下称，让双手自然放在膝盖或身体两侧。

## 呼吸引导

把注意力带到呼吸上——接下来我会用4-4-6呼吸法来引导你，让你找到呼吸的节奏，帮助你放松呼吸。

- **吸气**：轻轻地，通过鼻子缓缓吸气，心中默数 **1…2…3…4**，感受空气进入你的肺部，胸腔与腹部温柔地扩张。
- **屏息**：保持呼吸不动，默数 **1…2…3…4**，感受身体在平静与能量之间微微停顿。
- **呼气**：慢慢地，通过嘴巴或鼻子呼气，默数 **1…2…3…4…5…6**，让身体的紧张、杂念和疲惫随着气息一同流出。

继续这样的节奏，吸气、停顿、呼气，像潮水的涨落——

吸入时，想象你吸入清澈、温暖的空气；呼出时，想象你释放掉所有沉重与烦恼。

让我们再重复几次，感受身体的逐渐放松。感受一下自己的身体

## 感受自己

在这段艰难的时光，你遭遇了什么呢？

只需要简单的思考这件事情，不必过于纠结于这件事情，更不需要思考这件事情的因果关系或者如何改善。

你只需要作为一个旁观者，静静的看着这些事情像是天空的云朵一样浮过，一个想法接着一个想法，或者快，或者慢。

也许你很难做到这样。但，你也应该知道自己因为什么而那么难受。
